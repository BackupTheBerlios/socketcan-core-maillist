I can confirm that following settings for at91 are working with standard 
bitrates 50000, 100000, 125000, 250000, 500000 and 1000000:

static struct can_bittiming_const at91_bittiming_const = {
        .tseg1_min = 4,
        .tseg1_max = 16,
        .tseg2_min = 2,
        .tseg2_max = 8,
        .sjw_max = 4,
        .brp_min = 2,
        .brp_max = 128,
        .brp_inc = 1,
};

Settings for 1000000:
