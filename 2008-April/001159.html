Hi Adil,

Attached you'll find two patches:

at91sam9263-can-support.patch - Add support for CAN in
at91sam9263_devices.c and in the board support file for the 9263-EK
board.

socketcan-driver-at91.patch - The driver for AT91 CAN controllers.

ATM, this is all for AT91SAM9263 only because I don't have other AT91
boards with CAN. But I think it should be easy to support all AT91
devices that have CAN controllers on chip. Note that there are AT91
devices with more than one CAN controller. My current driver supports
only one, I noticed this too late, and it's sufficent for my SAM9263.

The best example how a socketcan driver should look like is Wolfgang
Grandeggers new version of the SJA1000 driver. You'll notice there's a
lot to do in my driver. Please feel free to rip things out, put in new
stuff - whatever you feel neccessary.

If you've got questions, please ask (me or the socketcan list). If
you've got something that can be tested, send it and I'll test it ASAP.

Good luck,
Hans
