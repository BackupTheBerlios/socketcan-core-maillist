Furthermore, socketcan is in the mainline kernel since 2.6.25-rc1, so
this is the "official" way how to deal with CAN under Linux.

Once you started using sockets for CAN, you'll see the many benefits it
has compared with simple char devices and wouldn't want to do it any
other way. You can use network analysis tools like wireshark for CAN as
well, for example.
