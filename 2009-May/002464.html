here are the patches for the "ip" program of the IPROUTE2 utility
suite adding support for CAN netlinks. They are against

  git clone git://git.kernel.org/pub/scm/linux/kernel/git/shemminger/iproute2

and require the Socket-CAN patch series:

  [PATCH v2-pre1 0/8] can: CAN network device driver interface and drivers

I cross compile them with:

  $ cd iproute2
  $ export DESTDIR=/data/ELDK/v4.2/ppc_6xx
  $ make CC=ppc_6xx-gcc

Please test and comment.
