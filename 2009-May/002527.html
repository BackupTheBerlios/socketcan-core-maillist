Hello,

the Socket-CAN branch "branches/netlink" with netlink support is now
available. I just committed the patch:

----------------
Netlink support with compatibility for old SYSFS interface and kernels

This patch adds netlink support to the Socket-CAN device driver
interface and brings the code in sync with the patch series:

[PATCH v3 0/7] can: CAN network device driver interface and drivers

posted to the netdev mailing list recently for kernel inclusion:

http://marc.info/?l=linux-netdev&m=124247730515356&w=2

It maintains backward compatibility for old kernel versions and
the deprecated SYSFS interface, which can be selected with the
option CONFIG_CAN_DEV_SYSFS.

While adapting the drivers, I realized various issues, which I marked
with "FIXME" in the header of the driver's source file, which should
be reviewed and fixed sooner than later.
----------------

I have not (yet) ported the old drivers (in drivers/net/can/old/).

Please give it a try.

Thanks,

Wolfgang.
