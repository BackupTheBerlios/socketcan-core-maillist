For now I have just a MPC8544 with an SJA1000 attached to the local bus.
Here is the git log:

This patch adds a driver for memory mapped SJA1000 device based
on OF description. Here is a FDT example node:

   can at 3,100 {
           compatible = "philips,sja1000";
           reg = <3 0x100 0x80>;
           clock-frequency = <8000000>:
           cdr-reg = <0x48>;
           ocr-reg = <0x0a>;
           interrupts = <2 0>;
           interrupt-parent = <&fpga_pic>;
   };

of-tree support for arm will come sooner than later ;-).
