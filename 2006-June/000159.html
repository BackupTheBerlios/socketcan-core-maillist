Hi Dave,

additionally to the PDF there is also a LaTeX source available.

Currently there are upcoming changes in the build process and how to install
the kernel-modules - and i do not know how this will end.

Btw. i can start to update the introduction and the Socket-API documentation
(raw, bcm, transport-protocol) in English. Just for your information before:

- raw-sockets: just a RAW-socket with filter options (ID-Filter(s),
Errorframe-Filter)
- bcm-sockets: Broadcast-Manager for (complex) Content-Filtering (e.g. on
cyclic messages)
- transport-protocol-sockets: create a point-to-point connection on a
CAN-bus
