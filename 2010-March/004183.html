Hello Friedrich,

even when you compiled the PEAK driver with netdev support, the configuration
of the bitrate is done via the PEAKs character device entries.

This is well documented in the

http://www.peak-system.com/fileadmin/media/linux/files/PCAN%20Driver%20for%20Linux_eng.pdf

see page 22, e.g. :

echo "i 0x4914 e" > /dev/pcan32

I did not try to configure the bitrates with the PEAK ioctl() interface so far.

Btw. why do you use 6.9b and not the latest 6.15??
