This patch adds the softing Gmbh CANcard to the socketCAN list.
Things you should know on this driver:
* Checkpatch.pl has a few warnings on complex macro's
* I tested with kernel 2.6.23.8
* It won't work on a big endian machine (yet)
* The drivers looses knowledge of which socket initiated which packet,
	so CAN_RAW_RECV_OWN_MSGS won't work
* status: worked for me,

In order to use the pcmcia card, you need firmware.
I didn't hear anything from Softing Gmbh yet, but noticed that the firmware in the form of byte-array
is released under gpl.
Therefore, after the real patches, a .tgz file will be added, with the code that I got the firmware with.
It does not work out of the box with newer drivers from softing, but the idea of what I did should be clear.
After this .tgz file, I'll just add the binary files

today, I am not able to run tests. But I don't expext issues, as nothing really changed since last week.
