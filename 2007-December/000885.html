this is the second version of my preliminary SYSFS support. It now uses the
prefix "can_" for the SYSFS files generated in "/sys/class/net/can*/" and
SYSFS support is properly initialized using the net device notifier callback.
We may also want to add "can_echo" and "can_debug" when it's clear how to
use local echo and debugging. You can find more doc in the header of the
patch.

Comments? Any objections to apply it?
