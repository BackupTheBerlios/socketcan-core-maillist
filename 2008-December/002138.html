> +##> Connect Tech Inc. CANpro/104
> +##> Options:
> +##>   debug - debug mask used to enable output classes
> +##>             0x00000001 - module parameters
> +##>             0x00000002 - resources
> +##>             0x00000004 - devices
> +##>             0x00000008 - IRQs
> +##>             0x00000010 - interrupt handling
> +##>           Warning: enabling the interrupt handling output class can
> +##>                    reduce system performance
> +##>   base  - base addresses of each controller
> +##>           (ensure J3.AddrMode is not populated)
> +##>   irq   - IRQ numbers of each controller
> +#alias can0 cti_canpro
> +#alias can1 cti_canpro
> +#options cti_canpro base=0x200,0x280 irq=10,5
