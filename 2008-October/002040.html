The driver can still be found in drivers/net/can/

Following these instructions you have a copy of my git repository.

>> git-clone
>> git://git.kernel.org/pub/scm/linux/kernel/git/torvalds/linux-2.6.git
>> cd linux-2.6/
>> git-remote add linux-can.mkl
>> git://pasiphae.extern.pengutronix.de/git/mkl/linux-can
>> git-fetch linux-can.mkl
>> git-branch -a
>> git-checkout linux-can.mkl/dev

With the last command you check-out top-off-tree of my (stable)
development. Because the at91can driver is not finished yet, although it
it stable it lacks some features like sending error frames into
userspace upon CAN errors. Anyhow, the at91 is in the mkl branch of my
repo, which can be obtained with:

git-checkout linux-can.mkl/mkl

The driver is located in drivers/net/can/at91_can.c, have a look at
arch/arm/mach-at91/board-sam9263ek.c how to include the device propperly
in you BSP.
