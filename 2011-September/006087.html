+  3.5 analysing CAN frames
+
+  Several tools for this task are provided along with can-utils:
+
+  - candump
+  - cansniffer
+
+  These command line tools show received CAN frames and have various
+  filter options.
+
+  Wireshark provides support to capture the whole traffic. The CAN
+  frames are dissected into frame type, flags and data sections. It's
+  basic CAN support provides basis for high protocol analysing filters
+  like CANopen, DeviceNet etc.
