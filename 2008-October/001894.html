The current SVN HEAD (828) has been converted into patches and these
patches are now in the dev branch.

http://pasiphae.extern.pengutronix.de/?p=mkl/linux-can;a=shortlog;h=refs/heads/dev

Please don't pull the whole tree from pengutronix, because it's still
behind a ADSL line, do sth. like this instead:

git-clone
git://git.kernel.org/pub/scm/linux/kernel/git/torvalds/linux-2.6.git
cd linux-2.6/
git-remote add linux-can.mkl
git://pasiphae.extern.pengutronix.de/git/mkl/linux-can
git-fetch linux-can.mkl
git-branch -a
git-checkout linux-can.mkl/dev
