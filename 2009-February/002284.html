Here comes my first and *preliminary* patch series to include
the CAN network device driver interface and some drivers into the
Linux kernel. It is based on the netdev GIT tree currently at
version 2.6.29-rc3:

  git://git.kernel.org/pub/scm/linux/kernel/git/davem/net-next-2.6

Please review and give it a try, send feedback, add your signed-off-by
or acked-by lines, if appropriate, check credits, check copyright notes,
etc...

There are still a few open issues:

- EXPORT_SYMBOLS with or without _GPL?
- Entries in the kernel's CREDITS and MAINTAINERS file.
- Some credits in the documentation.

Wolfgang.
