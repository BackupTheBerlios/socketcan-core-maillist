MSCAN Transmit Buffer Priority Register (TBPR)—MBAR + 0x0979

Description 0:7    PRIO[7:0]   Register defines local priority of
associated message buffer. Local priority is used for MSCAN internal
prioritization process and is defined to be highest for the smallest
binary number. MSCAN implements the following internal prioritization
mechanisms:
• All transmission buffers with a cleared TXEx flag
participate in prioritization immediately before start of frame (SOF)
is sent.
• Transmission buffer with lowest local priority field wins
prioritization.
• If more than one buffer has the same lowest priority,
message buffer with lower index number wins.
