This patch again adds the softing Gmbh CANcard to the socketCAN list.
 From a pragmatic point of view, I split the main big file in 2.

Things you should know on this driver:
* Checkpatch.pl has a few warnings on complex macro's
* I tested this version with kernel 2.6.23.8, previous prototypes
   have been tested with 2.6.26-rc6 & 7
* It won't work on a big endian machine (yet)
* In contrast to earlier prototypes,
   This time the drivers don't loose knowledge anymore of which socket initiated which packet,
     elementary CAN_RAW_RECV_OWN_MSGS use has been tested.
* status: worked for me,

In order to use the pcmcia card, you need firmware.
I didn't hear anything from Softing Gmbh yet, but noticed that the firmware in the form of byte-array
is released under gpl. This will get on the Berlios server soon since Softing has agreed to post it there.
