As expected.

You need to invoke

ip link set can0 type can bitrate 1000000
ip link set can1 type can bitrate 1000000


And to get this new 'ip' executable you need to build the iproute2 package
from the latest git repository:

git clone git://git.kernel.org/pub/scm/network/iproute2/iproute2.git

And then build the stuff and use the 'ip' in

iproute2/ip/ip

For a quick test i attached my latest 'ip' binary for x86 (compiled on my
Debian 5.0 system)

 md5sum ip.gz
56c484e9e42b1cc272c372f3d8a06576  ip.gz

But compiling open source is always better ;-)
